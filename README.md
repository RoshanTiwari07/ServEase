## 📖 About ServEase
**ServEase** is a modern **local service discovery & booking platform** that connects service providers directly with customers.  
From electricians to tutors, anyone can **create service ads, set pricing, share their portfolio, and manage customer inquiries** in real-time.  
Customers can **browse, filter, chat, and hire** with complete transparency.

**Tech Stack**:  
React Native • Tailwind CSS • Node.js (Express) • PostgreSQL • WebSockets

---

## 🚀 Features
- 📍 Location-based service discovery  
- 💬 Real-time in-app chat  
- 📂 Service provider profiles with work history  
- 💰 Transparent pricing display  
- 🔒 Secure user authentication
